# CryoSPARC Scripts for Sherlock

This directory contains all scripts needed to run CryoSPARC v4.7.1 on Stanford Sherlock.

## Directory Structure

scripts/  
├── slurm_templates/      # SLURM job submission templates  
└── management/           # Master node management scripts



## SLURM Templates

These are Jinja2 templates used by CryoSPARC to submit jobs to SLURM.
